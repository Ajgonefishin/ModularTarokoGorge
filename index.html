Create your own Digital Poetry - Taroko Gorge

Enter in lists of words, and this will create a procedurally generated, infinite poetry piece with them.
Modification by Ajani Mohanty. Click here to see my version of Taroko Gorge, inspired by neo-futuristic Japan.

Background

This project is a fork of the original Taroko Gorge website, which itself was a recreation of a Python program from 2009. The original program created infinite poetry based on lists of words that are reminiscent of Taroko Gorge National Park in Japan. With the introduction of the website, some people have made their own spins on the project by duplicating the source code and putting their own word lists in. This website aims to make this process a lot more streamlined for normal users, by providing fields to put their lists of words into and provide their own spin on this unique digital poetry project. If you'd like to see the original project or other peoples' versions, feel free to click the links on the right.

How-To

This website works off of lists of words that the program will use to create the poetry. There are 6 lists, using the following kinds of words: 'above' nouns, 'below' nouns, transitive verbs, intransitive verbs, imperative verbs, and adjectives.
Each text field has the default values in them, so feel free to edit as much or as little as you want. You can completely delete the default values if you'd like!
If it's confusing as to what kind of word you should put into a text field, look at the default values to see some examples of what works.
The program will automatically make nouns plural, so keep them singular when adding new ones.
Separate each entry with a comma followed by a space!
